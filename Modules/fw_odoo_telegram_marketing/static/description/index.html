Help for telegram Marketing


This module will be used to send message from odoo to telegram channel.


Installation

You must for beginning install SMS Marketing module.
To send message you must follow three steps.

First :


- go on telegram
- in the research bar typing "BotFather"


- click on the first link
- write "/start" as message
- the bot will respond a long message
- click on newbot (or respond "/newbot")

- type your bot name ("...bot")
- a bot will be create with a KEY number !
- if you don't rememeber the KEY number, type "/mybots"

Second:


- go on telegram
- click on the option (next to the research bar)


- click on "new channel"
- write a name for the channel
- go in the channel option (right top corner) :




- click on "manage channel"
- put the channel public (if it's not already done )
-create the link if it's not already done (choose a name that is not already taken)


- click on add administrator
- add your bot as administrator

Third:


- go on odoo
- go in apps and install fw_odoo_telegram_marketing
- go in Telegram Marketing
- click on create
- you are not required to fill in Dynamic Placeholder Generator
- go in the Telegram settings :


- write the name of your bot
- write the channels where you want to put your message (begin by "@")
- then write your text in the big field


- go in the Settings :

- you are required to fill the Name and the Medium

- click on the button "send now" or "Put in Queue"


That's it, you can see your message in all the telegram channel you send the message with Odoo
